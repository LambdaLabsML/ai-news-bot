# Lambda Newsy

![image](https://github.com/LambdaLabsML/newsy/assets/138706315/614149fd-badd-4008-8b14-542bc20356c7)

## Set up

```pip install -r requirements.txt```

### Required Environment Variables

- `SLACK_APP_TOKEN`
- `SLACK_BOT_TOKEN`
- `REDDIT_CLIENT_ID`
- `REDDIT_CLIENT_SECRET`
- `REDDIT_PASSWORD`
- `REDDIT_USERNAME`
- `OPENAI_API_KEY`

## Launch

```python3 app.py```
